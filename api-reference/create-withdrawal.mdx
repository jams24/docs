## Endpoint

```
POST /v1/merchant/withdrawal/create
```

## Authentication

This endpoint requires **two** authentication headers when called via API:

| Header | Required | Description |
|--------|----------|-------------|
| `X-API-Key` | Yes | Your merchant API key |
| `X-Withdrawal-Key` | Yes (API only) | Your withdrawal API key |

<Warning>
  The `X-Withdrawal-Key` header is required for all programmatic withdrawal requests (API, SDK, MCP agents).
  Dashboard withdrawals use 2FA instead and do not require this key.
</Warning>

### Getting a Withdrawal API Key

1. Go to **Dashboard → Settings → Withdrawal API Key**
2. Click **"Generate Key"**
3. Copy and securely store the key — it will only be shown once
4. Include it as `X-Withdrawal-Key` in your API requests

<Info>
  The withdrawal API key is separate from your main API key for added security. You can revoke it at any time
  from the dashboard to instantly disable all programmatic withdrawals.
</Info>

## Request Body

<ParamField body="amount" type="number" required>
  Amount to withdraw. Must be greater than the minimum for the selected currency.
</ParamField>

<ParamField body="currency" type="string" required>
  Cryptocurrency to withdraw. One of: `BTC`, `ETH`, `LTC`, `USDT_ERC20`, `USDT_BEP20`, `SOL`, `USDC_SOL`, `USDC_BEP20`
</ParamField>

<ParamField body="destinationAddress" type="string" required>
  External wallet address to send funds to. Must be valid for the selected currency.
</ParamField>

<ParamField body="description" type="string">
  Optional description or memo for the withdrawal (max 500 characters).
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.inventpay.io/v1/merchant/withdrawal/create \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "X-Withdrawal-Key: YOUR_WITHDRAWAL_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 100,
    "currency": "USDT_BEP20",
    "destinationAddress": "0x1E3D6848dE165e64052f0F2A3dA8823A27CAc22D",
    "description": "Monthly payout"
  }'
```

```javascript JavaScript
const response = await fetch(
  "https://api.inventpay.io/v1/merchant/withdrawal/create",
  {
    method: "POST",
    headers: {
      "X-API-Key": process.env.INVENTPAY_API_KEY,
      "X-Withdrawal-Key": process.env.INVENTPAY_WITHDRAWAL_KEY,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      amount: 100,
      currency: "USDT_BEP20",
      destinationAddress: "0x1E3D6848dE165e64052f0F2A3dA8823A27CAc22D",
      description: "Monthly payout",
    }),
  }
);

const data = await response.json();
console.log("Withdrawal ID:", data.data.withdrawalId);
```

```python Python
import requests

response = requests.post(
    "https://api.inventpay.io/v1/merchant/withdrawal/create",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "X-Withdrawal-Key": "YOUR_WITHDRAWAL_KEY",
        "Content-Type": "application/json",
    },
    json={
        "amount": 100,
        "currency": "USDT_BEP20",
        "destinationAddress": "0x1E3D6848dE165e64052f0F2A3dA8823A27CAc22D",
        "description": "Monthly payout",
    },
)

data = response.json()
print(f"Withdrawal ID: {data['data']['withdrawalId']}")
```

## Error Responses

### Missing Withdrawal API Key

```json
{
  "success": false,
  "error": "Withdrawal API key required for programmatic withdrawals",
  "message": "Provide X-Withdrawal-Key header. Generate one from Dashboard → Settings → Withdrawal API Key.",
  "code": "WITHDRAWAL_KEY_REQUIRED"
}
```

### Invalid Withdrawal API Key

```json
{
  "success": false,
  "error": "Invalid withdrawal API key",
  "code": "INVALID_WITHDRAWAL_KEY"
}
```

### Withdrawal Key Not Enabled

```json
{
  "success": false,
  "error": "Programmatic withdrawals are not enabled",
  "message": "Enable withdrawal API key from Dashboard → Settings to use this endpoint via API.",
  "code": "WITHDRAWAL_KEY_DISABLED"
}
```

### Insufficient Balance

```json
{
  "success": false,
  "error": "Insufficient Balance",
  "message": "Insufficient merchant balance. Available: 50 USDT_BEP20, Requested: 100 USDT_BEP20",
  "code": "INSUFFICIENT_BALANCE"
}
```

### Withdrawal Limit Exceeded

```json
{
  "success": false,
  "error": "Withdrawal Limit Exceeded",
  "message": "Withdrawal exceeds limits: daily limit (10000)",
  "code": "LIMIT_EXCEEDED"
}
```

### Invalid Address

```json
{
  "success": false,
  "error": "Invalid Destination Address",
  "message": "Invalid USDT_BEP20 address format",
  "code": "INVALID_ADDRESS"
}
```

## Withdrawal States

| Status | Description |
|--------|-------------|
| `PENDING` | Request created, awaiting processing |
| `PROCESSING` | Transaction submitted to blockchain |
| `COMPLETED` | Transaction confirmed, funds delivered |
| `FAILED` | Transaction failed, funds returned to balance |
| `CANCELLED` | Cancelled by merchant before processing |

## Notes

- The withdrawal amount is immediately locked from your available balance
- If a withdrawal fails, the locked amount is automatically returned to your available balance
- Network fees vary by blockchain congestion
- Service fee is 0.5% of the withdrawal amount
- You can cancel a `PENDING` withdrawal before it starts processing
