## Overview

Returns a paginated list of orders for your store. Optionally filter by order status.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your InventPay API key
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" optional>
  Page number (default: 1)
</ParamField>

<ParamField query="limit" type="integer" optional>
  Items per page, 1-50 (default: 20)
</ParamField>

<ParamField query="status" type="string" optional>
  Filter by status: `PENDING`, `PAID`, `PROCESSING`, `SHIPPED`, `COMPLETED`, `CANCELLED`, `REFUNDED`
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl "https://api.inventpay.io/v1/store/manage/orders?status=PAID&limit=10" \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript/TypeScript
const response = await fetch(
  "https://api.inventpay.io/v1/store/manage/orders?status=PAID&limit=10",
  {
    headers: { "X-API-Key": "YOUR_API_KEY" },
  }
);

const { data } = await response.json();
data.orders.forEach((order) => {
  console.log(`${order.orderNumber}: ${order.total} USDT`);
});
```

```python Python
import requests

response = requests.get(
    "https://api.inventpay.io/v1/store/manage/orders",
    headers={"X-API-Key": "YOUR_API_KEY"},
    params={"status": "PAID", "limit": 10}
)

for order in response.json()["data"]["orders"]:
    print(f"{order['orderNumber']}: {order['total']} USDT")
```

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 200 | Orders listed successfully |
