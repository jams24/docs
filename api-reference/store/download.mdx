## Overview

Retrieves the digital content for a completed order using the download token. This is a **public endpoint** — the customer uses the token they received after payment to access their purchased digital products.

The download token is a 64-character hexadecimal string, unique to each order.

<Info>
This endpoint is rate-limited to **10 requests per minute** per IP address. Each access is tracked (download count and last access time).
</Info>

## Path Parameters

<ParamField path="token" type="string" required>
  64-character hexadecimal download token from the order
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="order" type="object">
  Order information
  <Expandable title="order properties">
    <ResponseField name="orderNumber" type="string">
      Order number
    </ResponseField>
    <ResponseField name="storeName" type="string">
      Name of the store
    </ResponseField>
    <ResponseField name="paidAt" type="string">
      When payment was confirmed (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="items" type="array">
  Array of digital products
  <Expandable title="item properties">
    <ResponseField name="productName" type="string">
      Name of the product
    </ResponseField>
    <ResponseField name="productImage" type="string | null">
      Product image URL
    </ResponseField>
    <ResponseField name="fileUrl" type="string | null">
      Download URL for the file
    </ResponseField>
    <ResponseField name="fileName" type="string | null">
      Display name of the file
    </ResponseField>
    <ResponseField name="fileSize" type="number | null">
      File size in bytes
    </ResponseField>
    <ResponseField name="instructions" type="string | null">
      Setup or access instructions
    </ResponseField>
    <ResponseField name="licenseKey" type="string | null">
      License or activation key
    </ResponseField>
    <ResponseField name="textContent" type="string | null">
      Inline text content
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.inventpay.io/v1/store/download/a1b2c3d4e5f67890abcdef1234567890abcdef1234567890abcdef1234567890
```

```javascript JavaScript/TypeScript
const response = await fetch(
  `https://api.inventpay.io/v1/store/download/${downloadToken}`
);
const { order, items } = await response.json();

console.log(`Order: ${order.orderNumber}`);
items.forEach((item) => {
  console.log(`Product: ${item.productName}`);
  if (item.fileUrl) console.log(`Download: ${item.fileUrl}`);
  if (item.licenseKey) console.log(`License: ${item.licenseKey}`);
  if (item.instructions) console.log(`Instructions: ${item.instructions}`);
});
```

```python Python
import requests

response = requests.get(
    f"https://api.inventpay.io/v1/store/download/{download_token}"
)

data = response.json()
print(f"Order: {data['order']['orderNumber']}")

for item in data["items"]:
    print(f"Product: {item['productName']}")
    if item.get("fileUrl"):
        print(f"Download: {item['fileUrl']}")
    if item.get("licenseKey"):
        print(f"License: {item['licenseKey']}")
```

</CodeGroup>

## Error Responses

```json 404 - Invalid Token
{
  "error": "Invalid or expired download link"
}
```

```json 403 - Cancelled Order
{
  "error": "This order has been cancelled"
}
```

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 200 | Digital content returned |
| 403 | Order has been cancelled or refunded |
| 404 | Invalid or expired download token |
| 429 | Rate limit exceeded (max 10/min per IP) |
| 500 | Internal server error |
