---

## Overview

Upload unique keys, license codes, gift cards, or any per-customer content to a product's **key pool**. When a customer purchases the product, one key is automatically assigned (FIFO — first uploaded, first assigned) and delivered to them.

On the first upload, the product's key pool is automatically enabled and stock syncs to the available key count.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your InventPay API key
</ParamField>

## Path Parameters

<ParamField path="productId" type="string" required>
  The product ID to add keys to
</ParamField>

## Request Body

<ParamField body="keys" type="string[]" required>
  Array of unique keys/codes to add (1-10,000 per request). Duplicates within the product are automatically skipped.
  **Example:** `["XXXX-0001", "XXXX-0002", "XXXX-0003"]`
</ParamField>

<ParamField body="label" type="string" optional>
  Optional batch label for organizing keys (max 200 characters)
  **Example:** `"Batch #1"` or `"Premium Keys"`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data properties">
    <ResponseField name="added" type="integer">
      Number of new keys added
    </ResponseField>
    <ResponseField name="duplicates" type="integer">
      Number of duplicate keys skipped
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.inventpay.io/v1/store/manage/products/cm1prod123/keys \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "keys": [
      "LICENSE-AAAA-BBBB-0001",
      "LICENSE-AAAA-BBBB-0002",
      "LICENSE-AAAA-BBBB-0003"
    ],
    "label": "January Batch"
  }'
```

```javascript JavaScript
const response = await fetch(
  "https://api.inventpay.io/v1/store/manage/products/cm1prod123/keys",
  {
    method: "POST",
    headers: {
      "X-API-Key": "YOUR_API_KEY",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      keys: ["LICENSE-AAAA-BBBB-0001", "LICENSE-AAAA-BBBB-0002", "LICENSE-AAAA-BBBB-0003"],
      label: "January Batch",
    }),
  }
);

const result = await response.json();
console.log(`Added ${result.data.added} keys`);
```

```python Python
import requests

response = requests.post(
    "https://api.inventpay.io/v1/store/manage/products/cm1prod123/keys",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "keys": ["LICENSE-AAAA-BBBB-0001", "LICENSE-AAAA-BBBB-0002", "LICENSE-AAAA-BBBB-0003"],
        "label": "January Batch"
    }
)

result = response.json()
print(f"Added {result['data']['added']} keys")
```

</CodeGroup>

## Example Response

```json 201 - Created
{
  "success": true,
  "data": {
    "added": 3,
    "duplicates": 0
  },
  "message": "Added 3 keys (0 duplicates skipped)"
}
```

<Tip>
The product stock is automatically synced to the number of available keys. When keys are assigned or removed, stock updates accordingly.
</Tip>

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 201 | Keys added successfully |
| 400 | Invalid request (empty keys array, key too long) |
| 401 | Invalid or missing API key |
| 404 | Product or store not found |
| 429 | Rate limit exceeded |
