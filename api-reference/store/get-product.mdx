## Overview

Retrieves a single product from a public storefront by its slug. This is a **public endpoint** — no API key required. Only active products from active stores are returned.

Each product has a shareable URL: `https://inventpay.io/store/{storeSlug}/product/{productSlug}`

## Path Parameters

<ParamField path="slug" type="string" required>
  Store slug
</ParamField>

<ParamField path="productSlug" type="string" required>
  Product slug
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Product details
  <Expandable title="data properties">
    <ResponseField name="id" type="string">
      Product ID (used for checkout)
    </ResponseField>
    <ResponseField name="name" type="string">
      Product name
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL-safe product slug
    </ResponseField>
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    <ResponseField name="price" type="string">
      Price in USDT (decimal string)
    </ResponseField>
    <ResponseField name="images" type="string[]">
      Array of product image URLs
    </ResponseField>
    <ResponseField name="stock" type="string">
      `"in_stock"` or `"out_of_stock"`
    </ResponseField>
    <ResponseField name="variants" type="array">
      Product variant options
    </ResponseField>
    <ResponseField name="metadata" type="object">
      Custom product metadata
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.inventpay.io/v1/store/my-crypto-shop/products/claude-skills-guide
```

```javascript JavaScript/TypeScript
const response = await fetch(
  "https://api.inventpay.io/v1/store/my-crypto-shop/products/claude-skills-guide"
);

const { data } = await response.json();
console.log(`${data.name} - $${data.price} USDT`);
```

```python Python
import requests

response = requests.get(
    "https://api.inventpay.io/v1/store/my-crypto-shop/products/claude-skills-guide"
)

product = response.json()["data"]
print(f"{product['name']} - ${product['price']} USDT")
```

</CodeGroup>

<Note>
The public endpoint does not expose `digitalContent` — digital content is only accessible after payment via the [Download](/api-reference/store/download) endpoint.
</Note>

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 200 | Product returned |
| 404 | Store or product not found |
| 500 | Internal server error |
