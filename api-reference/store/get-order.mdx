---
title: "Get Order"
description: "Get detailed order information"
api: "GET /v1/store/manage/orders/{id}"
---

## Overview

Returns full details of a specific order, including line items, payment status, and customer information.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your InventPay API key
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  Order ID
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.inventpay.io/v1/store/manage/orders/cm1order789xyz \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript/TypeScript
const response = await fetch(
  "https://api.inventpay.io/v1/store/manage/orders/cm1order789xyz",
  {
    headers: { "X-API-Key": "YOUR_API_KEY" },
  }
);

const order = await response.json();
console.log(order.data.status);
```

```python Python
import requests

response = requests.get(
    "https://api.inventpay.io/v1/store/manage/orders/cm1order789xyz",
    headers={"X-API-Key": "YOUR_API_KEY"}
)

order = response.json()
print(order["data"]["status"])
```

</CodeGroup>

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 200 | Order retrieved successfully |
| 401 | Invalid or missing API key |
| 404 | Order or store not found |
| 429 | Rate limit exceeded |
