api: "GET /v1/store/manage"
---

## Overview

Returns the authenticated merchant's store details, including product count and configuration.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your InventPay API key
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Store details
  <Expandable title="data properties">
    <ResponseField name="id" type="string">
      Unique store identifier
    </ResponseField>
    <ResponseField name="merchantId" type="string">
      Your merchant ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Store name
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL-safe store slug
    </ResponseField>
    <ResponseField name="description" type="string">
      Store description
    </ResponseField>
    <ResponseField name="logo" type="string | null">
      Store logo URL
    </ResponseField>
    <ResponseField name="banner" type="string | null">
      Store banner URL
    </ResponseField>
    <ResponseField name="isActive" type="boolean">
      Whether the store is publicly visible
    </ResponseField>
    <ResponseField name="settings" type="object">
      Custom store settings
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      Creation timestamp (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.inventpay.io/v1/store/manage \
  -H "X-API-Key: YOUR_API_KEY"
```

```javascript JavaScript/TypeScript
const response = await fetch("https://api.inventpay.io/v1/store/manage", {
  headers: {
    "X-API-Key": "YOUR_API_KEY",
  },
});

const store = await response.json();
console.log(store.data.name);
```

```python Python
import requests

response = requests.get(
    "https://api.inventpay.io/v1/store/manage",
    headers={"X-API-Key": "YOUR_API_KEY"}
)

store = response.json()
print(store["data"]["name"])
```

</CodeGroup>

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 200 | Store retrieved successfully |
| 401 | Invalid or missing API key |
| 404 | No store found (create one first) |
| 429 | Rate limit exceeded |
