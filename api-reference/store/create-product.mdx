## Overview

Adds a new product to your store. Products are priced in **USDT** and automatically converted to the customer's chosen cryptocurrency at checkout. A URL slug is generated from the product name.

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your InventPay API key
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Product name (1-200 characters) **Example:** `"Premium API Access"`
</ParamField>

<ParamField body="description" type="string" optional>
  Product description (max 5000 characters) **Example:** `"Unlimited API calls for 30 days"`
</ParamField>

<ParamField body="price" type="number" required>
  Price in USDT (0.01 - 1,000,000) **Example:** `49.99`
</ParamField>

<ParamField body="images" type="string[]" optional>
  Array of HTTPS image URLs (max 10) **Example:** `["https://example.com/product.jpg"]`
</ParamField>

<ParamField body="stock" type="integer" optional>
  Available quantity. Omit for unlimited stock. **Example:** `100`
</ParamField>

<ParamField body="variants" type="array" optional>
  Variant options **Example:** `[{"name": "Plan", "options": ["Basic", "Pro"]}]`
</ParamField>

<ParamField body="metadata" type="object" optional>
  Custom key-value data attached to the product
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Product details
  <Expandable title="data properties">
    <ResponseField name="id" type="string">
      Unique product identifier
    </ResponseField>
    <ResponseField name="name" type="string">
      Product name
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL-safe product slug
    </ResponseField>
    <ResponseField name="price" type="string">
      Price in USDT (decimal string)
    </ResponseField>
    <ResponseField name="stock" type="integer | null">
      Available quantity or `null` for unlimited
    </ResponseField>
    <ResponseField name="isActive" type="boolean">
      Whether the product is visible
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      Creation timestamp (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.inventpay.io/v1/store/manage/products \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Premium API Access",
    "description": "Unlimited API calls for 30 days",
    "price": 49.99,
    "stock": 100,
    "images": ["https://example.com/product.jpg"]
  }'
```

```javascript JavaScript/TypeScript
const response = await fetch("https://api.inventpay.io/v1/store/manage/products", {
  method: "POST",
  headers: {
    "X-API-Key": "YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Premium API Access",
    description: "Unlimited API calls for 30 days",
    price: 49.99,
    stock: 100,
    images: ["https://example.com/product.jpg"],
  }),
});

const product = await response.json();
console.log(product.data.id); // Use this ID for checkout
```

```python Python
import requests

response = requests.post(
    "https://api.inventpay.io/v1/store/manage/products",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "name": "Premium API Access",
        "description": "Unlimited API calls for 30 days",
        "price": 49.99,
        "stock": 100,
        "images": ["https://example.com/product.jpg"]
    }
)

product = response.json()
print(product["data"]["id"])
```

</CodeGroup>

## Error Codes

| Status Code | Description |
| ----------- | ----------- |
| 201 | Product created successfully |
| 400 | Invalid request parameters |
| 401 | Invalid or missing API key |
| 404 | No store found (create a store first) |
| 429 | Rate limit exceeded |
